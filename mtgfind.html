MTGFind

Text 573-MTG-FIND (573-684-3463) or add it to a group chat to use MTGFind.

MTGFind takes the users prompt in the format [cardname]. It returns the first card alphabetically that fits that name. MTGFind searches scryfall, so anyone who knows how to use the Scryfall syntax can search for cards using that. For example, [o:"kraken" o:"whenever you draw a card" t:"enchantment"] will find Ominous Seas.
